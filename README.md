# Zeal Bot

AI-powered chatbot application built using React, Vite, Node.js, and OpenAI APIs.

---

## Features

* Real-time AI chat responses
* Responsive and modern UI
* Fast frontend powered by Vite
* REST API integration
* Secure environment variable handling
* Scalable frontend/backend structure

---

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js

### APIs

* OpenAI API

## Installation

Clone the repository:

```bash
git clone https://github.com/yashmittal7/zeal-bot.git
cd zeal-bot
```

Install dependencies:

```bash
npm install
```

If using separate frontend/backend folders:

```bash
cd frontend
npm install

cd ../backend
npm install
```

---

## Running the Project

Start frontend:

```bash
cd frontend
npm run dev
```

Start backend:

```bash
cd backend
npm run dev
```

---

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_OPENAI_API_KEY=your_api_key
PORT=5000
```
